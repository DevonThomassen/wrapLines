Tiny library (260B) to wrap lines after every "line break" in a paragraph.

Usage

wrapLines(container, openingtag, closingtag)

Notice:

This library only breaks each line! You can add your own animations with by example css || gsap. Openingtag && closingtag can be anything. Example:
var openingtag = '<div><span><section><thisiscoolhuh?>' // has to be a string btw;)
//Now do the same for the closing tag. Oh and please don't forget to close them correctly. You also have to make your own resize function maybe I add that in the futuru.
